# groc

## to run, try something like 

python3 run.py \ 
python3 render.py

### FYI

- start the world first with world.py
- run one of the world renderers next:
   - w-debug.py 
   - w-flat.py

### Supporting Files

groc.py - the Groc class \ 
world.py - the World class
